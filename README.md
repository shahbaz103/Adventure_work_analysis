# Adventure_work_analysis
#  Adventure Works End-to-End Data Analytics Project  
### By Shahbaz Ahmad

---

##  Project Overview

This project presents a complete end-to-end data analytics workflow using the Adventure Works dataset.

The objective was to transform raw transactional data into meaningful business insights using multiple analytics tools including SQL, Excel, Power BI, and Tableau.

This project simulates a real-world business scenario where raw sales data is extracted, cleaned, analyzed, and visualized to support data-driven decision-making.

---

##  Business Objectives

- Identify top-performing products and regions  
- Analyze revenue and profit trends  
- Evaluate customer purchasing behavior  
- Calculate key performance indicators (KPIs)  
- Provide strategic business recommendations  

---

##  Tools & Technologies Used

- **MySQL** – Data extraction, transformation, KPI calculations  
- **Microsoft Excel** – Data cleaning, pivot tables, trend analysis  
- **Power BI** – Interactive dashboard development  
- **Tableau** – Advanced analytics and storytelling  
- **GitHub** – Version control and project documentation  

---

##  Project Workflow

### 1️ Data Preparation (MySQL)

- Connected to Adventure Works database  
- Combined sales tables using UNION ALL  
- Calculated total sales and profit metrics  
- Aggregated customer and regional performance  
- Exported cleaned dataset for further analysis  

---

### 2️ Exploratory Data Analysis (Excel)

- Removed duplicates  
- Handled missing values  
- Standardized date and currency formats  
- Created pivot tables:
  - Sales by Product Category  
  - Year-over-Year growth by Region  
- Built trend visualizations  

---

### 3️ Interactive Dashboard (Power BI)

- Built data model and relationships  
- Created calculated measures and KPIs  
- Designed interactive dashboard with:
  - Total Profit  
  - Total Customers  
  - Total Orders  
  - Return Rate  
- Added slicers for dynamic filtering  

---

### 4️ Advanced Analytics (Tableau)

- Built quarterly sales comparison  
- Analyzed Sales vs Production Cost  
- Implemented dynamic filters and parameters  
- Compared Tableau capabilities with Power BI  

---

##  Key KPIs

- **Total Profit:** $12.08M  
- **Total Orders:** 28K  
- **Total Customers:** 18K  
- **Return Rate:** 6.26%  
- **Q4 Revenue Growth:** 30%  

---

##  Key Insights

- Revenue increased significantly in Q4 due to seasonal demand  
- Road Bikes are the top-selling product category  
- France shows strong revenue growth potential  
- Certain product segments have higher return rates  

---

##  Business Recommendations

- Launch targeted promotions in high-growth regions  
- Optimize inventory for high-demand products  
- Improve quality control to reduce return rate  
- Focus marketing strategy on top-performing categories
